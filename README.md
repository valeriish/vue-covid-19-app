# The Covid-19 Data Tracking Application

Resource: [API Data](https://covidtracking.com/data/api)

## Project setup
```
yarn install
```

### Compiles and minifies for production
```
yarn build:ssr
```

### Lints and fixes files
```
yarn lint
```
